# 使用说明
1.请先下载python3.8或以上版本安装进电脑。

2.请在右侧release区下载YANG SMS.zip文件，下载后自行解压。

3.打开文件内的YANG SMS.exe程序，按照操作指引进行。

4.请联系软件开发者YANG说明用途后获取YANG SMS Tonken许可密钥。

5.进入软件输入密钥后方可运行程序。
